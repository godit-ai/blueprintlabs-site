API Documentation

Integrate PMI's intelligent prompt compression into your applications in minutes. Our REST API is simple, fast, and designed for developers.

Overview

The PMI API provides a simple way to compress AI prompts while maintaining quality. All API requests are made to https://api.pmi.blueprintlabs.com/v1 and require authentication via API key.

💡 Base URL

All API requests should be made to: https://api.pmi.blueprintlabs.com/v1

Key Features

  • Compress prompts by 40-60% while maintaining 95%+ quality
  • Average response time of 1.43ms
  • Support for all major AI providers (OpenAI, Anthropic, Google)
  • Batch processing for high-throughput applications
  • Real-time quality scoring

Authentication

All API requests require authentication using your API key. Include your key in the X-API-Key header with every request.

Header Format 
X-API-Key: your_api_key_here

🔐 Security Best Practices

Never expose your API key in client-side code or public repositories. Use environment variables or a secrets manager to store your keys securely.

Getting Your API Key

1. Create an account or log in to your existing account

2. Navigate to your Dashboard

3. Copy your API key (or generate a new one)

Compress Prompt

Compress a single prompt using PMI's intelligent compression algorithms.

POST /compress

Compresses the provided prompt and returns the compressed version along with metadata.

Request Parameters

Parameter Type Description
promptrequired string The prompt text to compress. Maximum 10,000 characters.
leveloptional string Compression level: low, medium (default), or high
preserve_contextoptional boolean Whether to preserve semantic context. Default: true
target_modeloptional string Target AI model: gpt-4, claude, gemini, or generic (default)

Request Example

curl 
curl -X POST https://api.pmi.blueprintlabs.com/v1/compress \\
  -H "Content-Type: application/json" \\
  -H "X-API-Key: your_api_key_here" \\
  -d '{
    "prompt": "Please analyze this quarterly sales report and provide a comprehensive breakdown of performance metrics including revenue figures, quarter-over-quarter comparisons, and actionable recommendations.",
    "level": "medium",
    "target_model": "gpt-4"
  }'
Python 
import requests

api_key = "your_api_key_here"
url = "https://api.pmi.blueprintlabs.com/v1/compress"

payload = {
    "prompt": "Please analyze this quarterly sales report and provide a comprehensive breakdown of performance metrics including revenue figures, quarter-over-quarter comparisons, and actionable recommendations.",
    "level": "medium",
    "target_model": "gpt-4"
}

headers = {
    "Content-Type": "application/json",
    "X-API-Key": api_key
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()

print(f"Compressed: {data['compressed_prompt']}")
print(f"Tokens saved: {data['tokens_saved']}")
print(f"Quality score: {data['quality_score']}")
JavaScript 
const apiKey = 'your_api_key_here';
const url = 'https://api.pmi.blueprintlabs.com/v1/compress';

const payload = {
  prompt: 'Please analyze this quarterly sales report and provide a comprehensive breakdown of performance metrics including revenue figures, quarter-over-quarter comparisons, and actionable recommendations.',
  level: 'medium',
  target_model: 'gpt-4'
};

fetch(url, {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': apiKey
  },
  body: JSON.stringify(payload)
})
.then(response => response.json())
.then(data => {
  console.log('Compressed:', data.compressed_prompt);
  console.log('Tokens saved:', data.tokens_saved);
  console.log('Quality score:', data.quality_score);
})
.catch(error => console.error('Error:', error));

Response

200 OK - application/json 
{
  "success": true,
  "data": {
    "original_prompt": "Please analyze this quarterly sales report...",
    "compressed_prompt": "Analyze Q sales report: rev figs, QoQ compare, trends, actionable recs",
    "original_tokens": 87,
    "compressed_tokens": 46,
    "tokens_saved": 41,
    "compression_ratio": 0.471,
    "quality_score": 0.972,
    "processing_time_ms": 1.43,
    "estimated_savings_usd": 0.0012
  }
}

Response Fields

Field Type Description
success boolean Whether the compression was successful
data.original_prompt string The original prompt text
data.compressed_prompt string The compressed prompt text
data.original_tokens integer Token count of original prompt
data.compressed_tokens integer Token count of compressed prompt
data.tokens_saved integer Number of tokens saved
data.compression_ratio float Percentage of tokens reduced (0.0-1.0)
data.quality_score float Estimated quality retention (0.0-1.0)
data.processing_time_ms float Time taken to process in milliseconds
data.estimated_savings_usd float Estimated cost savings per request (USD)

Batch Compression

Compress multiple prompts in a single request for improved throughput.

POST /compress/batch

Compress up to 100 prompts in a single batch request.

Request Body

JSON 
{
  "prompts": [
    {
      "id": "req_001",
      "prompt": "First prompt to compress...",
      "level": "medium"
    },
    {
      "id": "req_002", 
      "prompt": "Second prompt to compress...",
      "level": "high"
    }
  ],
  "target_model": "gpt-4"
}

💡 Batch Processing Benefits

Batch requests have higher rate limits and reduced per-request overhead. Ideal for processing large datasets or high-throughput applications.

Error Codes

PMI uses standard HTTP response codes to indicate the success or failure of API requests.

400 Bad Request

The request was malformed or missing required parameters. Check the error message for details.

401 Unauthorized

Invalid or missing API key. Verify your API key is included in the X-API-Key header.

429 Too Many Requests

You've exceeded your rate limit. Wait before retrying or upgrade your plan for higher limits.

500 Internal Server Error

Something went wrong on our end. Please try again or contact support if the issue persists.

Error Response Format

Error JSON 
{
  "success": false,
  "error": {
    "code": "invalid_api_key",
    "message": "The provided API key is invalid or has been revoked.",
    "details": null
  }
}
